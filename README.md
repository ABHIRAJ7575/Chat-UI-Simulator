# Chat UI Simulator

A clean, dark-mode, WhatsApp-inspired chat UI simulator built using pure HTML, CSS, and vanilla JavaScript.  
This project focuses on realistic chat interactions, smooth UI behavior, and professional frontend structure without using any frameworks or backend services.

## Features
- Dark-mode only, clean and modern UI
- Mobile-first responsive layout
- Multiple chat threads
- Message persistence using LocalStorage
- Sent and received message bubbles with timestamps
- Simulated delivered and read indicators
- Typing indicator animation
- Keyboard support (Enter to send, Shift+Enter for new line)
- Smooth transitions and auto-scrolling

## Tech Stack
- HTML5
- CSS3 (CSS variables for theming)
- Vanilla JavaScript

## Project Structure
index.html
style.css
script.js

## Purpose
This project is designed to demonstrate practical UI/UX skills, frontend state handling, and clean code structure by simulating a real-world chat application experience.

## Setup
Clone the repository and open `index.html` in any modern browser. No build steps or dependencies required.
